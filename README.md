# Legally Legit AI

Australian legal document generator powered by AI.

## Quick Start

```bash
npm install
cp .env.example .env.local
# Add your API keys
npm run dev
```

## Features

- Employment Agreements
- Contractor Agreements  
- Privacy Policies
- Terms & Conditions
- AI-powered customization
- PDF generation
- Stripe payments

## Deployment

Push to GitHub and import to Vercel.
